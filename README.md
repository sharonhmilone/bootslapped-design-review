# Bootslapped — Design Review

Static HTML design review for the Bootslapped brand. Five pages:

- `index.html` — Homepage
- `article-email-list.html` — Why Your Email List Isn't Making You Money (Diagnostic)
- `article-email-platform.html` — How to Pick an Email Platform You Won't Migrate Off In a Year (Guide)
- `article-bookkeeping.html` — Wave vs. FreshBooks vs. QuickBooks (Comparison)

All pages link to each other via the nav and related articles sections.

## Brand decisions locked in these files

- Color: Brick (#C45C44) primary, Steel Teal (#4A9B8E) for Ask AI blocks
- Type: Oswald 700 wordmark, Barlow Condensed 700 headings, DM Mono body
- Layout: Editorial index, split-canvas hero, visible scaffolding, no card grids

Full spec: `bootslapped-style-guide.md` in the main project.
